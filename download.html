Installation Guide

Follow these simple steps to install the app on your device

1

Download APK

Click the download button above to download the APK file to your device.

2

Enable Unknown Sources

Go to Settings > Security > Enable "Unknown Sources" to allow APK installation.

3

Install App

Locate the downloaded APK file and tap to install the application.

4

Login & Enjoy

Open the app, login with your credentials, and start using all features.
